  • Sobre este Proyecto

Este proyecto busca crear una herramienta que ayude a los administradores de redes a gestionar y visualizar sus sistemas de manera más eficiente. Utilizando tecnologías web y de programación, como Python y Flask en el servidor, y HTML, CSS y JavaScript en el cliente, este proyecto tiene como objetivo desarrollar una plataforma que permita a los administradores de redes descubrir automáticamente la topología de sus redes, ofreciendo una interfaz visual interactiva para la configuración y monitoreo continuo del rendimiento de los dispositivos de red.
